Features

No bloat. No extensions.
Just your usage, always visible.

Built to do one thing well and get out of your way.

Auto-detects your browser

Reads existing session cookies from your browser. Nothing to configure -- it just works from the first launch.

Chrome Arc Firefox Brave Safari Edge

Claude + ChatGPT in one place

Session limits, weekly caps, Sonnet quotas, ChatGPT usage -- all in a single dropdown. Reset times in plain language.

5h session Weekly limits Reset times Notifications

Truly native macOS

~900 lines of Python. Runs as a LaunchAgent. Starts at login. Tiny footprint. No Electron, no bundled browser.

LaunchAgent No Electron MIT License

Setup

Three steps. Then forget about it.

The whole point is you should never have to think about this.

1

Run one command

Paste into Terminal. Python and all dependencies install automatically.

curl -fsSL .../install.sh | bash
2

Browser session detected

Reads your existing Claude.ai session from your browser. No login, no tokens.

3

Usage appears in menu bar

A color-coded percentage shows up immediately. Updates every 5 minutes.

Install

Up and running in seconds.

Two ways to install. Both take under a minute.

One-liner Recommended
$ curl -fsSL https://raw.githubusercontent.com/yagcioglutoprak/AIQuotaBar/main/install.sh | bash
Homebrew Tap
$ brew tap yagcioglutoprak/aiquotabar brew install aiquotabar aiquotabar &

Requires macOS 12+ and Python 3.10+. No Electron. No account needed.
