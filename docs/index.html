Never get blindsided by your Claude limit again

See your Claude.ai and ChatGPT usage limits live in your menu bar. Zero setup. One command.

curl -fsSL https://raw.githubusercontent.com/yagcioglutoprak/AIQuotaBar/main/install.sh | bash

Built for developers who hit limits

No Electron. No browser extension. Just a tiny Python script in your menu bar.

🔍

Zero setup

Reads cookies directly from Chrome, Arc, Firefox, Brave, Edge, or Safari. No copy-pasting tokens. No API keys for Claude.

📊

Claude + ChatGPT

Tracks Claude.ai session limits, weekly limits, and ChatGPT usage quotas in one place. Add OpenAI API spending too.



macOS native

~900 lines of Python. Runs as a LaunchAgent. Starts at login. Tiny memory footprint. No background services.
